# Case Trends Kenya

A full-stack e-commerce web app for Case Trends Kenya — your trusted electronics store in Nairobi.

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui + Framer Motion
- Supabase (Auth, PostgreSQL, Storage, Edge Functions)
- React Router v6

## Features
- Product catalog with category, brand, price, color, and stock filters
- Product detail pages with image gallery, reviews, and related products
- Cart system (localStorage for guests, Supabase DB for logged-in users, merged on login)
- Wishlist / Favorites (requires login)
- WhatsApp-based checkout with pickup or delivery options
- Admin panel: manage products, categories, hero slides, and users
- Background removal on product image upload via Edge Function

## Getting Started

```bash
bun install
bun dev
```
